# 前后端一体化打包和部署方案

## 📦 打包阶段

### 1. 后端打包
- 使用Maven打包Spring Boot应用
- 生成可执行的JAR文件
- 跳过测试以加快打包速度

### 2. 前端打包
- 使用npm run build打包Vue.js应用
- 生成静态资源文件
- 优化前端资源

### 3. 整合打包
- 将前端静态文件复制到Spring Boot的resources目录
- 创建统一的部署包
- 包含所有必要的配置文件

## 🚀 部署阶段

### 4. 服务器环境准备
- 安装JDK 17
- 安装Nginx
- 配置MySQL数据库
- 创建应用目录

### 5. 文件上传
- 使用SCP/SFTP上传整合包
- 上传配置文件
- 设置文件权限

### 6. Nginx配置
- 配置前端静态文件服务
- 配置后端API反向代理
- 配置HTTPS（可选）

### 7. 应用启动
- 启动后端Spring Boot应用
- 配置进程管理
- 配置日志输出

### 8. 部署验证
- 检查应用状态
- 测试前端访问
- 测试API接口

## 📋 需要创建的文件

1. **build-all.sh/bat** - 一体化打包脚本
2. **deploy.sh/bat** - 服务器部署脚本
3. **start.sh** - 应用启动脚本
4. **stop.sh** - 应用停止脚本
5. **nginx.conf** - Nginx配置文件
6. **application-prod.yml** - 生产环境配置
